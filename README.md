# Product Management Backend

##  Overview
Product Management Backend is a RESTful API built using Java and Spring Boot to manage products.  
It allows users to perform CRUD (Create, Read, Update, Delete) operations on product data efficiently.

This project demonstrates backend development concepts, clean architecture, and REST API implementation.

---

##  Features
- Create new products
- View all products
- Get product by ID
- Update product details
- Delete products
- Layered architecture (Controller, Service, Repository)

---

##  Tech Stack
- Java
- Spring Boot
- Maven
- REST APIs
- (Add Database: MySQL / H2 / PostgreSQL if used)
